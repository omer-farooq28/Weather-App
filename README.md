# ☁️ Weather App

A simple weather application built using **Python (Flask)**, **HTML**, **CSS**, and **JavaScript**. It provides real-time weather information for any city using the **WeatherAPI.com** API.

---

## Features

- Search weather by city
- Current temperature
- Weather condition
- Feels like temperature
- Humidity
- Wind speed
- Visibility
- UV Index
- Air Quality (PM2.5)
- Responsive user interface
- Default city: Hyderabad

---

## Technologies Used

- Python 3.14
- Flask
- HTML5
- CSS3
- JavaScript
- WeatherAPI.com

---

## Installation

### Clone the repository

```bash
git clone https://github.com/your-username/weather-app.git
```

### Go to the project folder

```bash
cd weather-app
```

### Install dependencies

```bash
pip install -r requirements.txt
```

---

## Configure API Key

Create a `.env` file in the project root.

```env
WEATHER_API_KEY=YOUR_API_KEY
```

Get your free API key from:

https://www.weatherapi.com/

---

## Run the Application

```bash
python app.py
```

Open your browser and visit:

```
http://localhost:3000
```
